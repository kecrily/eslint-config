# @kecrily/eslint-config

> A ESLint config for Percy.

## Use

Install eslint, config and typescript:

```sh
pnpm i eslint @kecrily/eslint-config typescript -D
```

Add `eslintConfig` to `package.json`:

```json
{
  // ...
  "eslintConfig": {
    "extends": "@kecrily"
  }
}
```
